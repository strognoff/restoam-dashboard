## Configure app URLs
Set via `.env` or environment variables:

```

VITE_LOCATIONS_URL=http://localhost:5174
VITE_WORKORDERS_URL=http://localhost:5175
VITE_ADMIN_USER=admin
VITE_ADMIN_PASS=admin
```

## Run

```bash
npm install
```
